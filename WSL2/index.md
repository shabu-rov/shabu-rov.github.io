# WSL2

![logo](/WSL2/res/WSL2.png)

Windows Subsystem for Linux (WSL) — слой совместимости для запуска Linux-приложений в ОС Windows 10. 

Позволяет полноценно работать с ROS из под Windows в подсистеме Ubuntu Linux 16.04.
